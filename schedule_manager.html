🔧 Firebase 初期設定
初回のみ設定が必要です。設定は このブラウザに保存され、次回以降は自動接続します。
1 Firebaseプロジェクトを作成する
① console.firebase.google.com を開く　② 「プロジェクトを追加」　③ プロジェクト名を入力して作成
2 Realtime Database を有効にする
左メニュー「構築」→「Realtime Database」→「データベースを作成」→ ロケーション選択 →「テストモードで開始」を選択
3 セキュリティルールを設定する
Realtime Database の「ルール」タブを開き、以下のルールを貼り付けて「公開」ボタンを押す
{ "rules": { ".read": true, ".write": true } }
4 アプリを登録して設定値を取得する
プロジェクト設定（⚙アイコン）→「マイアプリ」→「</>」(Web)ボタン → アプリ登録 → 表示された firebaseConfig の各値を以下に入力
